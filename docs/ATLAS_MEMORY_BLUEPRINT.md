# Memory Blueprint
Version: 1.0.0

---

# Mission

Project ATLAS exists to discover, validate, and continuously refine every measurable
pregame factor that influences baseball outcomes.

ATLAS does not attempt to imitate sportsbooks.

ATLAS attempts to understand baseball.

The objective is to estimate the true probability of baseball outcomes through
evidence-driven learning.

---

# Core Philosophy

ATLAS never asks:

    Does this matter?

ATLAS asks:

    WHEN does it matter?

    WHY does it matter?

    AGAINST WHO does it matter?

    UNDER WHAT CONDITIONS does it matter?

    BY HOW MUCH does it matter?

    HOW CONFIDENT are we?

Evidence answers every question.

# Long Term Goal

ATLAS does not attempt to beat sportsbooks directly.

ATLAS attempts to understand baseball more completely than any sportsbook.

If ATLAS estimates true probabilities more accurately than the market,
positive expected value becomes a natural consequence.

---

# Final Principle

Every engine added to ATLAS must answer four questions.

1.

What knowledge does this engine create?

2.

Where does that knowledge live?

3.

How will another engine use it?

4.

How will ATLAS learn whether that knowledge is correct?

If an engine cannot answer these questions,
it does not belong inside ATLAS.
