# ATLAS Constitution

## Core Rule
ATLAS provides the facts. The models discover what matters.

## Model Separation
No single universal ATLAS score.

Separate models:
- Win / Moneyline
- Totals / Over-Under
- Player props
- Live win probability
- Learning engine

## Analysis Order
1. ATLAS data
2. Identity engine
3. Matchup engine
4. Pregame feature matrix
5. Separate models
6. Market comparison
7. Postgame learning
